# Marketplace Platform API

REST API для управления платформой маркетплейса. Реализовано на **FastAPI** с использованием **Pydantic** (валидация схем) и **SQLAlchemy** (асинхронный PostgreSQL).

## Предметная область (Вариант 15)

- **Продавцы** — регистрация и управление профилями продавцов
- **Товары** — каталог товаров с категориями и остатками
- **Заказы** — оформление заказов с автоматическим списанием со склада
- **Отзывы** — отзывы покупателей с пересчётом рейтинга продавца
- **Комиссии** — начисление и учёт комиссий платформы

## Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка БД

Убедитесь, что PostgreSQL запущен и доступна база `marketplace_platform`.

Параметры подключения (можно переопределить через переменные окружения с префиксом `MP_`):

| Параметр      | Значение                  |
|---------------|---------------------------|
| Сервер        | 127.0.0.1                 |
| Порт          | 5432                      |
| Пользователь  | mp_user                   |
| Пароль        | 12345                     |
| База данных   | marketplace_platform      |

### 3. Запуск

```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 4. Документация API

После запуска:

- **Swagger UI**: http://127.0.0.1:8000/docs
- **ReDoc**: http://127.0.0.1:8000/redoc

## API Endpoints

### Продавцы (`/api/v1/sellers`)
| Метод   | Путь              | Описание                    |
|---------|-------------------|-----------------------------|
| POST    | `/api/v1/sellers/` | Создать продавца            |
| GET     | `/api/v1/sellers/` | Список продавцов            |
| GET     | `/api/v1/sellers/{id}` | Продавец по ID           |
| PATCH   | `/api/v1/sellers/{id}` | Обновить продавца        |
| DELETE  | `/api/v1/sellers/{id}` | Удалить продавца         |

## Обработка ошибок

Приложение использует кастомные исключения (`MarketPlaceError`), которые автоматически преобразуются в JSON-ответы с соответствующим HTTP-статусом:

| Статус | Тип ошибки                           |
|--------|--------------------------------------|
| 404    | Сущность не найдена                  |
| 409    | Конфликт (дубликат email, нет стока) |
| 400    | Некорректные данные                  |
